---
title: "Delete Affiliate"
api: "DELETE /v1/affiliates/{id}"
description: "Delete an affiliate user"
---

<Warning>
  This action is irreversible. All associated data including referrals and commissions will be permanently deleted.
</Warning>

<ParamField path="id" type="string" required>
  The unique affiliate ID to delete
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.affonso.io/v1/affiliates/aff_123abc" \
  -H "Authorization: Bearer sk_live_your_api_key"
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "data": {
    "id": "aff_123abc",
    "deleted": true
  }
}
```
</ResponseExample>
