---
title: "List Affiliates"
api: "GET /v1/affiliates"
description: "Get a paginated list of all affiliates"
---

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Number of results per page (max 200)
</ParamField>

<ParamField query="partnershipStatus" type="string">
  Filter by partnership status: `PENDING`, `APPROVED`, `REJECTED`
</ParamField>

<ParamField query="search" type="string">
  Search by name or email
</ParamField>

<ParamField query="groupId" type="string">
  Filter by affiliate group ID
</ParamField>

<ParamField query="programId" type="string">
  Filter by affiliate program ID
</ParamField>

<ParamField query="sort" type="string">
  Sort field and direction (e.g., `createdAt:desc`, `partnershipStatus:asc`)
</ParamField>

<ParamField query="expand" type="string">
  Expand related data: `promoCodes`, `commissionOverrides`, `invoiceDetails`, `payoutMethod`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.affonso.io/v1/affiliates?limit=10&status=active" \
  -H "Authorization: Bearer sk_live_your_api_key"
```
</RequestExample>
