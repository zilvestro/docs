---
title: "Update Affiliate"
api: "PUT /v1/affiliates/{id}"
description: "Update an existing affiliate"
---

<ParamField path="id" type="string" required>
  The unique affiliate ID
</ParamField>

<ParamField body="name" type="string">
  Update the affiliate's display name
</ParamField>

<ParamField body="email" type="string">
  Update the affiliate's email address
</ParamField>

<ParamField body="groupId" type="string">
  Move affiliate to a different group (set to null to remove from group)
</ParamField>

<ParamField body="companyName" type="string">
  Update company name for invoicing (set to null to clear)
</ParamField>

<ParamField body="countryCode" type="string">
  Update two-letter ISO country code for invoicing (set to null to clear)
</ParamField>

<ParamField body="payoutMethod" type="string">
  Update payout method: `paypal`, `wise`, `payoneer`, `paxum`, `webmoney`, `skrill`, `wire_transfer`, `bitcoin`, `ethereum`, `alipay`, `wechat`, `manual`, `none`
</ParamField>

<ParamField body="payoutDetails" type="object">
  Update payout method details (email, accountId, address, bank details depending on method)
</ParamField>

<ParamField body="status" type="string">
  Update partnership status: `PENDING`, `APPROVED`, `REJECTED`
</ParamField>

<RequestExample>
```bash cURL
curl -X PUT "https://api.affonso.io/v1/affiliates/cmj8pptm10002p3010z8mk9ak" \
  -H "Authorization: Bearer sk_live_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "companyName": "John LLC",
    "countryCode": "US",
    "payoutMethod": "paypal",
    "payoutDetails": {
      "email": "john@example.com"
    },
    "status": "APPROVED"
  }'
```
</RequestExample>
