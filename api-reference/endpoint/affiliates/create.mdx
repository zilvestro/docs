---
title: "Create Affiliate"
api: "POST /v1/affiliates"
description: "Create a new affiliate user"
---

<ParamField body="name" type="string" required>
  The affiliate's display name
</ParamField>

<ParamField body="email" type="string" required>
  The affiliate's email address
</ParamField>

<ParamField body="programId" type="string" required>
  The affiliate program ID to assign the affiliate to
</ParamField>

<ParamField body="groupId" type="string">
  Optional affiliate group ID to assign to
</ParamField>

<ParamField body="companyName" type="string">
  Optional company name for invoicing
</ParamField>

<ParamField body="countryCode" type="string">
  Optional two-letter ISO country code for invoicing
</ParamField>

<ParamField body="payoutMethod" type="string">
  Payout method: `paypal`, `wise`, `payoneer`, `paxum`, `webmoney`, `skrill`, `wire_transfer`, `bitcoin`, `ethereum`, `alipay`, `wechat`, `manual`, `none`
</ParamField>

<ParamField body="payoutDetails" type="object">
  Payout method details (email, accountId, address, bank details depending on method)
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.affonso.io/v1/affiliates" \
  -H "Authorization: Bearer sk_live_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Smith",
    "email": "jane@example.com",
    "programId": "cm7xutqbb0001yfkcrnpextmp",
    "companyName": "Jane LLC",
    "countryCode": "US",
    "payoutMethod": "paypal",
    "payoutDetails": {
      "email": "jane@example.com"
    }
  }'
```
</RequestExample>
