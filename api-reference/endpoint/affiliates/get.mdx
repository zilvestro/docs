---
title: "Get Affiliate"
api: "GET /v1/affiliates/{id}"
description: "Retrieve a single affiliate by ID with optional expanded data"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the affiliate to retrieve
</ParamField>

## Query Parameters

<ParamField query="expand" type="string">
  Comma-separated list of related data to include in the response. Valid values: `promoCodes`, `commissionOverrides`, `invoiceDetails`, `payoutMethod`. You can combine multiple values: `promoCodes,payoutMethod`.
</ParamField>

## Response

The response includes the affiliate object with the following fields:

<ResponseField name="id" type="string">
  Unique identifier for the affiliate
</ResponseField>

<ResponseField name="name" type="string">
  The affiliate's display name
</ResponseField>

<ResponseField name="email" type="string">
  The affiliate's email address
</ResponseField>

<ResponseField name="tracking_id" type="string">
  Unique tracking ID for the affiliate
</ResponseField>

<ResponseField name="source" type="string">
  Source of the affiliate creation (e.g., `"api"`, `"dashboard"`)
</ResponseField>

<ResponseField name="partnership_status" type="string | null">
  Partnership status with the program: `"PENDING"`, `"APPROVED"`, `"REJECTED"`, or `null` if no partnership exists
</ResponseField>

<ResponseField name="onboarding_completed" type="boolean">
  Whether the affiliate has completed their onboarding process
</ResponseField>

<ResponseField name="program_id" type="string">
  The affiliate program ID the affiliate is assigned to
</ResponseField>

<ResponseField name="group_id" type="string | null">
  The affiliate group ID if assigned, otherwise `null`
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the affiliate was created
</ResponseField>

### Expanded Fields (when using `expand` parameter)

<ResponseField name="promoCodes" type="array" expandable>
  Array of promo codes associated with the affiliate. Each item contains:
  - `code` (string): The promo code
  - `promo_code_id` (string | null): Internal promo code ID
  - `coupon_id` (string | null): Coupon ID if linked
  - `discount_type` (string | null): Type of discount (e.g., `"percentage"`, `"fixed"`)
  - `discount_value` (number | null): Discount value
  - `duration` (string | null): Duration of the discount (e.g., `"forever"`, `"once"`)
  - `created_at` (string): ISO 8601 timestamp
</ResponseField>

<ResponseField name="commissionOverrides" type="array" expandable>
  Array of commission overrides for the affiliate. Each item contains:
  - `id` (string): Override ID
  - `incentive_id` (string): Incentive ID
  - `incentive_name` (string): Name of the incentive
  - `incentive_type` (string): Type of incentive (e.g., `"commission"`)
  - `amount` (number): Commission amount
  - `is_percentage` (boolean): Whether the amount is a percentage
  - `apply_to` (string): What the override applies to (e.g., `"all_plans"`)
  - `product_ids` (array): Array of product IDs if applicable
</ResponseField>

<ResponseField name="invoiceDetails" type="object | null" expandable>
  Invoice details for the affiliate. Contains:
  - `company_name` (string | null): Company name
  - `first_name` (string): First name
  - `last_name` (string): Last name
  - `address_line_1` (string): Street address
  - `address_line_2` (string | null): Additional address line
  - `city` (string): City
  - `postal_code` (string | null): Postal/ZIP code
  - `state` (string | null): State/Province
  - `country` (string): Two-letter country code
  - `vat_id` (string | null): VAT identification number
  - `tax_id` (string | null): Tax identification number
</ResponseField>

<ResponseField name="payoutMethod" type="object | null" expandable>
  Payout method configuration. Contains:
  - `type` (string): Payout method type (e.g., `"paypal"`, `"wire_transfer"`)
  - `details` (object): Method-specific details (varies by type)
</ResponseField>
