---
title: "Get Affiliate"
api: "GET /v1/affiliates/{id}"
description: "Retrieve a single affiliate by ID"
---

<ParamField path="id" type="string" required>
  The unique affiliate ID
</ParamField>

<ParamField query="expand" type="string">
  Expand related data: `promoCodes`, `commissionOverrides`, `invoiceDetails`, `payoutMethod`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.affonso.io/v1/affiliates/aff_123abc" \
  -H "Authorization: Bearer sk_live_your_api_key"
```
</RequestExample>
