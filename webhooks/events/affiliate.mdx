## Overview

Affiliate events are triggered when changes occur to affiliate accounts in your program. Use these webhooks to sync affiliate data with your CRM, send welcome emails, or trigger onboarding workflows.

## Events

<AccordionGroup>
  <Accordion title="affiliate.created" icon="user-plus">
    Triggered when a new affiliate signs up for your program or is manually created by a program owner.

    ```json
    {
      "id": "evt_abc123",
      "type": "affiliate.created",
      "createdAt": "2024-01-15T10:30:00.000Z",
      "data": {
        "affiliateId": "aff_xyz789",
        "affiliateProgramId": "prog_abc123",
        "trackingId": "john-doe",
        "status": "PENDING",
        "email": "john@example.com",
        "name": "John Doe"
      }
    }
    ```
  </Accordion>

  <Accordion title="affiliate.confirmed" icon="circle-check">
    Triggered when an affiliate confirms their email address.

    ```json
    {
      "id": "evt_abc123",
      "type": "affiliate.confirmed",
      "createdAt": "2024-01-15T10:30:00.000Z",
      "data": {
        "affiliateId": "aff_xyz789",
        "affiliateProgramId": "prog_abc123",
        "trackingId": "john-doe",
        "status": "ACTIVE",
        "email": "john@example.com",
        "name": "John Doe"
      }
    }
    ```
  </Accordion>

  <Accordion title="affiliate.updated" icon="pen">
    Triggered when an affiliate's details are updated, including status changes.

    ```json
    {
      "id": "evt_abc123",
      "type": "affiliate.updated",
      "createdAt": "2024-01-15T10:30:00.000Z",
      "data": {
        "affiliateId": "aff_xyz789",

        "trackingId": "john-doe",

        "name": "John Doe"
      }
    }
    ```
  </Accordion>

  <Accordion title="affiliate.deleted" icon="trash">
    Triggered when an affiliate is removed from your program.

    ```json
    {
      "id": "evt_abc123",
      "type": "affiliate.deleted",
      "createdAt": "2024-01-15T10:30:00.000Z",
    ```

  </Accordion>
</AccordionGroup>

## Payload Fields

| Field | Type | Description |
|-------|------|-------------|
| `affiliateId` | string | Unique identifier for the affiliate |
| `affiliateProgramId` | string | The program this affiliate belongs to |
| `trackingId` | string | The affiliate's unique tracking/referral code |
| `status` | string | Current status: `PENDING`, `ACTIVE`, `INACTIVE` |
| `email` | string | Affiliate's email address |
| `name` | string \| null | Affiliate's display name |
